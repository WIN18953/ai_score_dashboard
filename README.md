# 🎓 AI Student Score Prediction Dashboard

This project is a Machine Learning dashboard built with Streamlit.

## 📌 Features
- Predict student exam scores
- Compare Linear Regression and Polynomial Regression
- Interactive charts
- Download prediction results as CSV

## 🛠 Technologies Used
- Python
- Streamlit
- Scikit-learn
- Pandas
- Plotly

## 🚀 How to Run

1. Install dependencies:
   pip install -r requirements.txt

2. Run the app:
   streamlit run app.py

## 📂 Project Structure
project-folder/
│
├── app.py
├── requirements.txt
├── README.md
└── dataset.csv

## 🤖 Machine Learning Model

This project uses:

- Linear Regression
- Polynomial Regression

The model is trained using Scikit-learn and evaluated using:
- Mean Absolute Error (MAE)
- R² Score
